# Parens
Implement an algorithm to print all valid (e.g., properly opened and closed) combinations of n pairs of parentheses.
EXAMPLE
Input: 3
Output: ((())), (()()), (())(), ()(()), ()()()

# Solutions:

# Swift
```
O(n!)
```

# Installation
Run `make` to build and run Swift files. Or you can run `.playground` file.

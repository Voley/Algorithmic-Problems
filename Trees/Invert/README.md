# Invert Binary Tree

Invert a binary tree.
     4
   /   \
  2     7
 / \   / \
1   3 6   9
to
     4
   /   \
  7     2
 / \   / \
9   6 3   1

# Solutions:

# Swift
```
O(n)
```
# C++
```
O(n)
```

# Installation
Run `make` to build and run C++ files and run Swift files. Or you can run `.playground` file.

# Merge Intervals
Given a collection of intervals, merge all overlapping intervals.

For example,
Given [1,3],[2,6],[8,10],[15,18],
return [1,6],[8,10],[15,18].



# Solutions:

# Swift
```
O(n)
```
# C++
```
O(n)
```

# Installation
Run `make` to build and run C++ files and run Swift files. Or you can run `.playground` file.
